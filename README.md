# Pokedex Guide Website
This website is a rokedex guide. Search button toggle was used, when clicked the input opens. When we type the pokemon name we want to search one by one, it does filtering.

# Technologies
<ul>
	<li>HTML: Used to create the page structure.</li>
	<li>CSS: Used for styling and responsive design.</li>
 	<li>JAVASCRIPT: Used to create interactive web pages.</li>
</ul>
